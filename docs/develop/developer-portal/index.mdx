# Using Band Developer Portal

## What is Band Devloper Portal

Band Developer Portal is a web application that help developers to build and deploy your own BandChain v3 testnet price feed within a few minutes.

## How to use Band Developer Portal

1. Go to [Band Developer Portal](https://developer.bandprotocol.com)
2. Click on the **"Sign In"** button
3. Sign in with your Google or Github account
4. You will be redirected to the developer portal

# Band CLI & REST Endpoints

You can get the RPC and REST endpoints from [here](/develop/api-endpoints).

## Get Data Source by ID

**CLI**: `bandd query oracle data-source [id] --node ${RPC}`

**Path**: `/api/oracle/v1/data_sources/{id}`
**Method**: `GET`

```bash
$ bandd query oracle data-source 1 --node ${RPC}
$ curl -X GET "${REST}/oracle/v1/data_sources/1"
{
  "data_source": {
    "owner": "band1m5lq9u533qaya4q3nfyl6ulzqkpkhge9q8tpzs",
    "name": "d1",
    "description": "d1",
    "filename": "c56de9061a78ac96748c83e8a22330accf6ee8ebb499c8525613149a70ec49d0",
    "treasury": "band1m5lq9u533qaya4q3nfyl6ulzqkpkhge9q8tpzs",
    "fee": [
      {
        "denom": "uband",
        "amount": "1"
      }
    ]
  }
}
```

## Get Active Status of a Validator

**CLI**: `bandd query oracle validator [validator-address] --node ${RPC}`

**Path**: `/api/oracle/v1/validators/{validator-address}`
**Method**: `GET`

```bash
$ bandd query oracle validator bandvaloper1kfj48adjsnrgu83lau6wc646q2uf65rf84tzus --node ${RPC}
$ curl -X GET ${REST}/oracle/v1/validators/bandvaloper1kfj48adjsnrgu83lau6wc646q2uf65rf84tzus
{
  "status": {
    "is_active": true,
    "since": "2021-09-07T14:52:54.598085215Z"
  }
}
```
